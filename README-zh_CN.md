## 使用

我们推荐使用 `yarn` 去管理依赖. 如果你还没有安装 `yarn`, 请参考下列命令:

```
npm install -g yarn
```

或者, 请参考 [yarn 安装指南](https://yarnpkg.com/en/docs/install). 现在，我们只要下载代码就可以开始你的工作了

```
git clone -b 3.x https://github.com/kaysonwu/react-app.git
```
